Основы Python

Что такое Python?
Python — активно развивающийся язык программирования. Его очень просто понять и изучить. Практически все программы на этом языке быстро пишутся, так как написание кода не займет у вас много времени. Вам точно стоит попробовать этот язык, потому что вашу первую работающую программу вы сможете написать сразу после первого урока.
Плюсы и минусы Python
Один из главных плюсов, так это то, что выучить синтаксис Python не сложно. Язык построен так, чтобы на нем было легко писать, а написанный код было легко читать. Еще на Python можно написать практически что угодно (настольные приложения, игры, скрипты по автоматизации, комплексные системы расчёта, системы управления жизнеобеспечением и многое многое другое) без ощутимых проблем.
Но также у Python есть ряд своих минусов. Один из главных это его скорость, потому что при разработке программ с повышенным требованием к эффективности, Python может значительно проигрывать по скорости своим статическим братьям (C/C++, Java, Go) Вторым не менее важным фактором является динамическая типизация. На первый взгляд она может показаться райским наслождением, потому что не надо заморачиваться и присваивать значения переменной, потом еще переприсваевать и так далее. Но есть и обратная сторона медали. С ростом вашего кода, следить за типом передаваемых друг другу данных бывает очень сложно. И еще один минус в том, что при копировании кода с другого ресурса, в некоторых случаях, он может скопироваться без сохранения отступов. Поэтому код будет неправильным, а вам придется долго добавлять табуляцию в каждую строчку.
Но не смотря на все эти проблемы Python остается лидирующим языком програмирования на данный момент.
Кто использует Python?
